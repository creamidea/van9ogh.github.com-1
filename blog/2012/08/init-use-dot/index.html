C language:精确初始化结构体的一种新方法

Written by Liu Peng at Monday, 27 August 2012 - 00:00

[PS] Press keyboard [->]<-] to read previous/next artical.

以前

大多时候在定义的时候初始化一个结构体我们都是这么做的.

struct student {
	char x, y, z;
	int h;
	char name[20];
};

struct student stu = 
{
	{
		'a',
		'b',
		'c',
		10,
		"LiuPeng"
	},
};

现在

现在有一种新方法，我们可以精确初始化某一个部分， 或者跳过一些变量的初始化.

struct student {
	char x, y, z;
	int h;
	char name[20];
};

struct student stu = 
{
	{
		.x = 'a',
		.y = 'b',
		.z = 'c',
		.h = 10,
		.name = "LiuPeng"
	},
};

就是这样， 惊奇吧!
